ESP32 Lexical Analyzer Compiler

This project utilizes an ESP32 microcontroller to create a simple web server that allows users to upload a file through a web interface. Once the file is uploaded, the server performs lexical analysis on its content using a DFA (Deterministic Finite Automaton) to identify different types of tokens such as keywords, identifiers, numbers, strings, and symbols.

  • File Upload: Users can upload a file through a web interface.
  • Lexical Analysis: Analyzes the uploaded file and identifies tokens using a DFA.
  • Result Display: Displays the analysis results in a formatted HTML table.
  • Web Server: ESP32 hosts a web server for easy access to the analyzer.

üîß Installation

  1. Clone the repository: 
    git clone https://github.com/Esmail-sarhadi/Esp32-Lexical-Analyzer-compiler.git
  2. Open the project in Arduino IDE:
    • Navigate to the cloned directory and open the main.ino file.
  3. Configure Wi-Fi credentials:
    • Update the ssid and password variables with your Wi-Fi network information.
  4. Upload the code to ESP32:
    • Select the correct board and port in the Arduino IDE.
    • Click "Upload".

üöÄ Usage

  1. Connect to the ESP32 web server:
    • Open the Serial Monitor in Arduino IDE to get the ESP32's IP address.
    • Enter this IP address in a web browser.
  2. Upload a file:
    • Use the file upload form on the web page.
  3. View analysis results:
    • The lexical analysis results will be displayed in a table format on the web page.

The main components of the code include:

  • setup(): Initializes Wi-Fi and web server.
  • loop(): Handles incoming client requests.
